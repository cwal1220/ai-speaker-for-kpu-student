
## AI Speaker for KPU Student

## Function
  1. Snowboy를 이용한 음성호출
  2. STT(Speech To Text), TTS(Text To Speech)
  3. 웹 크롤링을 이용한 공지사항 알림
  4. GPS 기반 날씨 알림
  5. 셔틀버스 시간 알림
  6. 구내식당 메뉴 추천 기능
  
## Hardware
  1. Raspberry PI 3 Model B
  2. Raspberry PI Public 7' Touch Screen
  3. USB Microphone
  
## Tools
  1. Python3
  2. MariaDB
  3. PyQt5
  
## APIs
  1. Google Cloud Speech API
  2. Naver TTS API
  3. SK planet Weather API
